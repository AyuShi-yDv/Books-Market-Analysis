## 📌 Project Overview

This project is an end-to-end data analysis pipeline that extracts book data from an online bookstore using web scraping techniques and transforms it into meaningful insights through an interactive dashboard.

It demonstrates the complete workflow of:
**Data Collection → Data Cleaning → Data Analysis → Data Visualization**

---

## ⚙️ Features

- 🕷️ Multi-page web scraping using BeautifulSoup  
- 📊 Extraction of book titles, prices, and ratings  
- 🧹 Data cleaning and preprocessing using Pandas  
- 📈 Interactive dashboard using Streamlit  
- 🔍 Filtering based on ratings  
- 📉 Price and rating distribution analysis  

---

## 🛠 Tech Stack

- **Python**
- **Pandas**
- **BeautifulSoup**
- **Matplotlib**
- **Streamlit**

---

## 📊 Key Insights

- Distribution of book prices across multiple pages  
- Analysis of rating patterns across products  
- Identification of high-rated books  
- Dataset built with 100+ records using multi-page scraping
